BuildR Demo 3.0 Readme
=================

Thanks for checking BuildR available in full from the Unity Asset Store (https://www.assetstore.unity3d.com/#!/content/82220).
A fast, simple and effective system to create buildings within Unity.

How To
======
Create a building.
To get started, go to Unity Menu > Tools > BuildR Cities > Create New Building
This creates a new building gameobject in your scene â€“ select this.
The menu allows you to start making a building by adding a volume. Click on "Add New Volume"
You can now either click and drag a rectangle in your scene, or click place points until you complete a valid shape.
